发表于 2018-05-09 | 分类于 算法
k_means聚类算法python实现，并对50个基因表达量聚类

k means 聚类

思路

K-means算法以k为参数，把n个对象分为k个簇，使簇内具有较高的相似度，而簇间的相似度较低。

  1. 随机选择K个点作为初始的聚类中心
  2. 对于剩下的点，根据其与聚类中心的距离，将其归入最近的簇
  3. 对每个簇，计算所有点的均值作为新的聚类中心
  4. 重复步骤2,3直到聚类中心不再变化
