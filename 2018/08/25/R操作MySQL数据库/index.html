R操作MySQL数据库

  • 连接MySQL数据库

    conn <- dbConnect(MySQL(), 
    dbname="r_data",
    username="root", 
    password="root", 
    host="127.0.0.1", 
    port=3306)

summary(conn)

  • 查看数据库信息

    dbGetInfo(conn)

  • 查看数据库表格列表

    dbListTables(conn)

  • 从R导出数据到数据库

dbWriteTable函数用来将R中的数据导入MySQL数据库，其包括创建新的表格，覆盖已有的表格，追加数据到表格。用法为dbWriteTable(MySQL连接对象名，表格名，R数据框名)

(1) 创建

> dbWriteTable(conn,"test",cars)
[1] TRUE
> dbListTables(conn)
[1] "test"

（2） 覆盖

dbWriteTable(conn,"test",cars2,overwrite = TRUE)

（3）追加

dbWriteTable(conn,"test",cars3,append = TRUE)

（4）删除表格

dbRemoveTable(con,"test")
  • 导入数据到R数据框

dbReadTable()函数用来从MySQL数据库提取整个表格（前提是表格数据量不是很大）。

dbReadTable(conn,"test")

  • 获取表格字段列表

    dbListFields(conn,"test")

  • 查询语句使用

dbGetQuery()函数用来提取表格中指定的行(使用LIMIT查看前几行)

dbGetQuery(conn,"SELECT * FROM test LIMIT 5;")

另一种方式是使用dbSendQuery()查询数据库，然后使用dbFetch()获取数据

query <- dbSendQuery(conn,"SELECT * FROM test;")
data <- dbFetch(query,n=5)

  • dbGetInfo()函数也可以查看query对象的信息

    dbGetInfo(query)

  • dbGetStatement()函数用于返回query的MySQL查询语句

    > dbGetInfo(query)
$`statement`
[1] "SELECT * FROM test;"

  • dbGetRowCount()函数用于返回dbFetch对象查询到的数据行数

    >dbGetRowCount(data)
5

  • dbColumnInfo()函数用于查看query对象的列信息

    > dbColumnInfo(query)
       name    Sclass      type length
1 row_names character BLOB/TEXT  65535
2     speed    double    DOUBLE     22
3      dist    double    DOUBLE     22

  • dbClearResult()函数释放dbSendQuery()函数的数据库资源

    dbClearResult(query)

  • 检查数据类型

    > dbDataType(RMySQL::MySQL(),"A")
[1] "text"
> dbDataType(RMySQL::MySQL(),1:5)
[1] "bigint"
> dbDataType(RMySQL::MySQL(),1.5)
[1] "double"

  • 断开数据库连接

    dbDisconnect(conn)
